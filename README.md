# Christian-Nils' CV

- Icons using fontawesome
- Need to install the following packages before compiling:
  - `texlive-roboto`
  - `texlive-moresize`
  - `texlive-fontawesome5`
  - `texlive-wrapfig`
  - `texlive-biblatex-publist`
- Compile as follows:

```
pdflatex -> biber -> pdflatex X 2
```
